## 💡 核心理念

### agent协作开发的本质
不是让AI替代你编程，而是让AI成为你最好的项目启动助手：
- **自然表达**：用人话说需求，让AI推断技术细节
- **渐进澄清**：在对话中逐步明确和完善想法
- **快速验证**：小步快跑，及时调整方向

### ADHD友好设计原则
- **微会话**：每次专注5-15分钟，符合注意力特点
- **低摩擦**：减少决策疲劳，提供明确指引
- **即时反馈**：看得见的进步，保持动力
- **结构化**：模板和流程化，降低启动门槛
